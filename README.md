# SillyTavern Persistent Itemized Prompts

Do you hate that you lose your prompt information if you move browser? Well NOT ANYMORE!

This extension will persist your prompts on your disk so if you change your browser or if you run SillyTavern as a
service, you can actually get your prompt information all the time.

### This is a companion extension for plugin https://github.com/Enerccio/SillyTavern-PersistentItemizedPrompts-plugin 

There is more information about it in the plugin github page.

## Installation

1. Instal plugin (check it's README.md for more information)
2. Install the extension in ST using the github link: https://github.com/Enerccio/SillyTavern-PersistentItemizedPrompts
3. On first run it will synchronize itemizedPrompts into a local file
4. ???
5. Profit - this extension requires no manual use
